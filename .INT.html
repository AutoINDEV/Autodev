Create the mind in .INT Framework

Indev Inc is a company founded  by Saúl Alejandro  :  a powerful rendering engine fully integrated with a complete set of intuitive tools and rapid workflows to create interactive 3D and 2D content in  INDEX; easy multiplatform publishing; thousands of quality, ready-made assets in the Asset Store and a knowledge-sharing community.

For independent developers and studios, INDEV’s democratizing ecosystem smashes the time and cost barriers to creating uniquely beautiful games. They are using INDEXto build a livelihood doing what they love: creating games that hook and delight players on any platform.
